# Prompts de LariTechFarms

Esta carpeta contiene prompts personalizados para el proyecto LariTechFarms.

## Uso

Para usar un prompt, escribe `@prompt nombre-del-prompt` en el chat de Amazon Q.

## Prompts disponibles

- `componente-angular`: Genera componentes Angular con la estructura del proyecto
- `servicio-api`: Crea servicios para consumir APIs
- `formulario-reactivo`: Genera formularios reactivos con validaciones
