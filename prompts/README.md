# Prompts para LariTechFarms

Esta carpeta contiene prompts organizados para el desarrollo del proyecto Angular.

## Estructura:
- `components/` - Prompts para crear componentes
- `features/` - Prompts para funcionalidades específicas  
- `design/` - Prompts relacionados con diseño y UI
- `templates/` - Plantillas de prompts reutilizables

## Uso:
Copia el contenido del prompt y úsalo con Amazon Q para generar código consistente con el estándar del proyecto.
